The insertBefore() method in JavaScript is used to insert a new element (or node) before a reference node in the DOM.

The syntax is as follows:- parentNode.insertBefore(newNode, referenceNode);

    Parameters:
  • parentNode: The parent element to which the new element will be added.
  • newNode: The new node (element, text, etc.) that you want to insert into the DOM.
  • referenceNode: The existing node that you want the new node to be inserted before. If this node is null, the new node is added at the end of the parent node's child nodes.
