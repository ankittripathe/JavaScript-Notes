Synchronous:- stored in Main stack

Asynchronous: stored in Side stack

Async-Await:- async function always return a promise

Syntax:- async function myFunc(){....}

Await:- A keyword used to pause the execution of an async function until the Promise is settled (either resolved or rejected).

    Benefits of Using Async-Await:-
  • (1) Async-Await ka use hum krte hai Asynchronous ko Synchronous ki tarah Behave krwane ke liye.
  • (2) With Async-Await aap Synchronous code bhi aise likh skte ho jaise ki aap noraml Synchronous code likhe rahe ho.
  • (3) Await ka matalb hai agli line tab ka na execute kro jabtak agle ka answer na aa jaye.
  • (4) Await ko work krwane ke liye Await ke parent function me Async lagana padega.
