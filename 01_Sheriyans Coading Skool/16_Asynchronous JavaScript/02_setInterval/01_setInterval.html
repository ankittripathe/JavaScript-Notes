setInterval:- Is a JavaScript function that repeatedly executes a specified callback function at a fixed time interval (in milliseconds).

The execution continues until it is explicitly stopped using clearInterval.

Syntax:- setInterval(callback, interval, ...args);

let intervalID = setInterval(callback, delay, [arg1, arg2, ...]);

setinterval:- setinterval ka code kuch der baad chalta hai baar baar ek particular interval time mein.
