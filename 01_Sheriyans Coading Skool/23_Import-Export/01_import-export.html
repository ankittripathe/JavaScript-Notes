Export:- Exports make parts of a module (functions, objects, or variables) available for use in other modules. There are two types of exports:
  • (1) Named Exports:- You explicitly define what you want to export. Multiple named exports can exist in a single module
  • (2) Default Exports:- A module can have only one default export. Typically used when the module exports a single main value

    • Import:-You can use import to bring functions, objects, or variables from one module into another.
  • (1) Importing Named Exports:- Use curly braces {} to import named exports. You can import multiple values.
  • (2) Importing All Named Exports:- You can import everything into a single object.
  • (3) Importing Default Exports:- No curly braces are needed for default exports. You can also rename the imported default.
  • (4) Mixing Named and Default Imports:- You can combine both named and default imports.
